# STUDENT-PERFORMANCE-REPORT
Interactive Excel Dashboard Project
## Project Overview
This project presents an interactive Excel dashboard analyzing how lifestyle, behavioral, and academic factors influence student performance.
The objective was to identify key drivers of academic success and understand the relationship between attendance, study habits, mental health, and entertainment usage.

## Objectives
- Analyze the impact of attendance on exam performance.
- Evaluate how study hours affect academic outcomes.
- Examine the relationship between mental health and performance.
- Assess the influence of entertainment consumption on result.
- Identify patterns among high-performing students.

## Tools & Techniques Used
- Microsoft Excel
- Pivot Tables
- Slicers (Interactive Filters)
- Data Grouping 
- Trend Analysis
- Dashboard Design & Visualization

## Dashboard Features
- Interactive filters (Gender, Result Status)
- Attendance vs Performance comparison
- Mental Health impact analysis
- Entertainment usage vs score trend
- Lifestyle pattern visualization

## Key Insights
- Students with attendance above 90% achieved the highest average scores.
- Higher mental health ratings showed a positive correlation with exam performance.
- Academic performance declined as daily entertainment hours increased.
- Balanced study and sleep patterns contributed to improved outcomes.

## Analytical Approach
- Cleaned and structured percentage-based attendance data
- Created grouped categories for attendance, study hours, and mental health
- Built Pivot Tables to calculate average exam scores by category
- Developed interactive dashboard with slicers
- Extracted data-driven insights from trends and comparisons
